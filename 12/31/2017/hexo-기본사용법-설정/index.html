Dec 31, 2017 in blog, hexo

Hexo 기본사용법, 설정

지난 포스팅을 통해서 기본적인 설치를 마쳤다면 로컬에서 Hexo 서버를 띄워서 확인 가능합니다.
지난 포스팅 : Git + Hexo + Theme + Atom

로컬에서 확인하기

hexo가 설치된 곳으로 이동해서 작업한다.

서버를 구동한 뒤 http://localhost:4000 으로 확인한다.

1. 기본사용법

포스팅 작성하기

post_title에는 포스팅의 제목을 적으면 된다.

기본 포스팅들이 저장되는 위치는 /source/_posts 폴더에 저장된다.

포스팅이 작성이 되면 로컬에서도 바로 확인이 가능하고 이상이 없으면 github에 배포를 한다.

배포과정은

  • github에 배포하기위해 로컬에 배포할 정적인 파일을 생성한다.
  • github로 배포한다.

한줄로도 가능하다.

배포를 했는데 잘 반영이 안될 경우는 로컬에 있는 정적인 파일을 한번 지우고 다시 generate을 한번 해 본다.
