Hexo Plugin SEO

Jan 25, 2018 in blog, hexo

Hexo Plugin을 이용해서 SEO에 도전하자.

Theme에서도 SEO에 친화적이라고 소개를 하고 있지만 100% 완벽하지는 않다. 그래서 plugin을 이용하든지 세팅을 수정하든지 해서 좀 더 SEO 친화적이게 변경해야 된다.
자! 그럼, 하나씩 plugin을 install하여 수정해 보도록 하자.

hexo plugin은 아래에서 확인할 수 있다.
꼭 SEO에 관련된 것 외에도 필요한 부분은 추가해서 사용하면 좋을듯 하다.

Plugin을 이용한 SEO

Plugin : https://hexo.io/plugins/

위치는 테마소스 내부에 layout\_partial\header.ejs 파일을 수정하면 된다.
파일을 열고 아래로 내려오면 태그가 보인다. 안 쪽에 아래 코드를 삽입하면 된다.

그리고 브라우져 개발자도구에서 삽입이 된 것을 확인할 수 있다.

사이트맵 설정

사이트맵으로 사이트의 구성을 체계화 시켜놓으면 검색봇 입장에서는 수 많은 페이지를 크롤링할 때 상당한 비용을 줄일 수 있을 뿐더러 사이트를 빠르고 효율적으로 크롤링이 가능하게 된다.

Plugin : hexo-generator-seo-friendly-sitemap

RSS 피드 설정

최근에 작성한 포스트에 대한 정보를 가지고 있는 RSS 설정되 해보도록 하자.

Plugin : hexo-generator-feed

설치하기

소스 수정하기
테마소스 안에 있는 테마환경파일인 _config.yml 파일에 다음 항목을 추가하면 된다.

nofollow 설정

검색봇에게 사이트의 외부링크를 링크로 인식 못하게 만드는 것이다. 하지만 내부링크는 상관없다.
페이지에 rel="noopener" target="_blank" 속성이 자동으로 추가되어 모든 외부링크에 적용된다.

Plugin : hexo-autonofollow

설치하기

소스 수정하기
테마소스 안에 있는 테마환경파일인 _config.yml 파일에 다음 항목을 추가하면 된다.

true : 활성화 한다는 의미, default는 false이다.

설정이 잘 되었는지 확인은 구글 웹마스터 도구에서 확인이 가능하다.

다음 포스팅 구글웹마스터 사이트 등록에서 확인해보자.

robots.txt 설정

검색봇에게 내 사이트를 크롤링 해 갈지 말지의 여부, 크롤링의 범위등을 설정해 놓으면 검색봇은 설정한 옵션에 따라 크롤링하게 된다.

Plugin : hexo-generator-robotstxt
