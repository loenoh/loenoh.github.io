Hexo 구글애드센스(Google Adsense) 추가

Feb 07, 2018 in hexo

Hexo 블로그에 Google Adsense를 추가해볼려고 합니다.

구글애드센스를 추가한다는 것은 즉, 블로그에 광고를 삽입하겠다는 뜻입니다.

마찬가지로 광고삽입 전에 구글애드센스에 가입이 되어야하고 가입과정은 생략하겠습니다.

사이트 : 구글애드센스(Google Adsense)


전체적인 작업진행

  1. Google Adsense 광고 만들기
  2. 코드 생성
  3. 광고가 나오될 위치에 해당 코드 삽입


구글애드센스(Google Adsense) 광고 만들기

구글애드센스 사이트에 로긴 후 홈으로 이동합니다.

내광고 > 콘텐츠 > 광고단위 > 새광고단위 를 클릭해서 광고를 만듭니다.


텍스트 및 디스플레이 광고를 선택합니다.


이름, 광고크기, 광고유형등을 선택하고 저장 및 코드생성을 클릭합니다.

광고크기는 특별한 경우가 아니면 페이지에 따라 변화되는 반응형을 추천합니다.




코드가 생성된 팝업 화면이 나타납니다.

코드를 복사하셔도 되고 광고단위 목록에서도 코드를 다시 가져올 수 있습니다.



그럼 광고가 만들어졌습니다.

그럼 광고가 나올 곳을 찾아 붙여 보겠습니다.


코드 삽입

hexo의 파일 구조를 보면 테마폴더/layout/layout.ejs 파일이 전체적인 구조에 관련된 파일이고, 세부적인 구조들은 테마폴더/layout/_partial폴더안에 각각 ejs파일로 구성되어져 있다.

그래서 구글 애드센스 코드를 바로 삽입하는 것이 아니라 ejs파일을 생성하고 생성된 파일을 광고가 나오게 될 부분에 삽입하는 형태를 추천합니다.

그럼 테마폴더/layout/_custom_ads 폴더를 하나 생생하고 google_adsense.ejs파일을 생성해서 해당파일에 구글 애드센스에서 생성한 코드를 넣어줍니다.

이제 광고가 삽입될 부분에는 다음과 같은 코드 한줄을 추가하면 적용이 됩니다.


본문 위, 아래 추가

본문에 삽입할 경우는 layout.ejs파일을 수정하면 되고, 파일은 일부만 발췌했습니다.

파일위치 : 테마폴더/layout/layout.ejs



윗 부분을 수정하면 메인 화면 포스팅 위, 아래에도 추가되고 클릭했을 때 해당 포스팅에도 추가되어집니다.
하지만 메인에는 보이지 않고 각각의 포스팅에만 적용할 경우에는 post.ejs파일을 수정하면 가능합니다.

파일위치 : 테마폴더/layout/_partial/post.ejs


사이드바 하단부분 추가

사이드바 하단에 추가는 개인적으로는 보기가 좀 그래서 추가는 안했던 부분인데 필요하신 분은 추가하시면 될 것 같습니다.

수정파일위치와 파일내용은 아래와 같고, 파일내용은 길이 관계로 일부만 발췌했습니다.

파일위치 : 테마폴더/layout/_partial/sidebar.ejs

Hexo Tranquilpeak Theme Disqus적용

Jan 31, 2018 in hexo

Hexo 블로그에서 Disqus를 사용해보자.

Disqus는 블로그 포스트 하단에 댓글을 달 수 있게 해주는 기능이다.

먼저 Disqus 사이트에서 가입을 해야한다.

가입을 한 후 Home > setting > Admin > General 메뉴에 가면 다음 화면이 나타난다.

Add버튼을 클릭하면 댓글을 달 사이트를 등록하는 화면이 나온다.

사이트를 만들고 나면 disqus를 설치할 사이트가 어떤 플랫폼인지 선택하라는 페이지가 나온다.
해당 플랫폼이 없으면 하단부에

I don't see my platform listed, install manually with Universal Code. 라는

버튼을 클릭하면 소스에 직접 삽입할 수 있는 코드가 나타난다.

Hexo블로그는 플랫폼에는 없지만 왼쪽 Configure Disqus로 바로 넘어가도 무방하다.

Website Name, Website URL, Category, Description, Language를 세팅하고 마치면 된다.

세팅이 끝난 후 Admin 페이지에 가보면 Shortname 이 보일 것이다.
이것은 unique하고 변경도 되지 않는다. 만약 다른것을 사용하고 싶으면 다시 추가해야 된다.

그러면 Hexo 블로그 테마 환경 설정 파일 _config.yml에 추가하면된다.

disqus_shortname: [추가된 해당 사이트의 shortname]

이렇게 세팅하고 포스팅 하단을 확인하면 댓글을 달 수 있는 부분이 추가되어 있을 것이다.

GitHub Hexo Theme Atom 설치

Dec 29, 2017 in hexo

Blogging Platform 구축

Atom 에디터로 작성한 마크다운 문서를 GitHub Page + Hexo를 이용하여 Blog 구축 하기

들어가며 : Blog 선택

Atom 에디터로 작성한 마크다운 문서를 GitHub Page를 이용하여 Blog운영하기
여지껏 여러 블로그를 사용해보고 테스트도 해 본 것 같다.
국민 블로그인 네이버, 티스토리, 워드프레스, 블로거등을 사용하다보니 뭔가 모른 아쉬움이 있었다.
github를 이용해서도 사용해볼까 하다가 이런 저런 이유로 미뤄왔다.
문서는 마크다운으로 작성을 하긴했었는데 블로그에 접목을 안하고 있었는데, 이제서야 필요성을 느끼고 접목을 할려고하다보니 기존에 사용하던 블로그들은 안되는건 아니지만 중간에 한 단계를 거쳐야 하는 등의 불편함이 있었다.
그래서 github를 이용해서 블로그를 운영하기로 했다.
프로그램에 대한 이해도가 조금 있어야 건드릴 수 있는 부분이지만, 요즘은 많은 문서들이 한글화도되어져 있고 잘 나와져 있어서 무리는 없을 것 같다.

무작정 설치하고 사용하기 전에 운영에 전반적인 그림을 그려봤다.
블로그를 운영할 때 포스팅하고 퍼블리싱하는 과정을 플랫폼화 시키고 싶었다.
서칭을 해보니 많은 사람들이 jekyll를 이용하고 있었고 Hexo로도 퍼블리싱을 많이 하는 것 같았다.
무슨 심보인지는 모르지만, 좀 더 덜 사용하는 Hexo를 이용해보기로 하고 세팅에 들어갔다.
작업은 다음의 순서로 설치와 세팅을 진행했다.

Git -> Hexo -> Theme -> Atom

기본적이고 간단한 설치는 이번 포스팅에서, 세부적인 부분은 다음 포스팅에서 소개하겠습니다.

Git 설치

https://git-scm.com/download
해당 사이트로 이동해서 운영체제에 맞게 설치한다.

Hexo 설치

https://hexo.io/
hexo 를 설치한다.

https://hexo.io/ko/
한글화된 문서도 제공하고 있다.

Theme 설치

hexo를 설치하면 기본테마(landscape)가 설치된다.

다른 테마를 설치하기 위해서는 https://hexo.io/themes/ 접속한다.
접속한 사이트에 있는 여러테마 중에 마음에 드는 테마를 골라서 설치하면 된다.
테마를 선택하면 설치문서와 데모를 볼 수 있다. 자세한 테마설치는 다음 포스팅에서 확인해본다.

Atom 설치

마크다운 에디터로 atom을 이용할 예정이다.

  1. atom 설치 : https://atom.io/

  2. git, hexo 관련 패키지 설치 ctrl + ,(comma) 단축키로 setting에 들어간 후 install에서 패키지를 검색해서 install한다.

  • git-plus
  • atom-hexo
  • hexo-front-matter
