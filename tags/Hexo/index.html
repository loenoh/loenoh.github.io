Theme 설치

hexo를 설치하면 기본테마(landscape)가 설치된다.

다른 테마를 설치하기 위해서는 https://hexo.io/themes/ 접속한다.
접속한 사이트에 있는 여러테마 중에 마음에 드는 테마를 골라서 설치하면 된다.
테마를 선택하면 설치문서와 데모를 볼 수 있다. 자세한 테마설치는 다음 포스팅에서 확인해본다.

Atom 설치

마크다운 에디터로 atom을 이용할 예정이다.

  1. atom 설치
    https://atom.io/
    여기서 다운로드해서 설치한다.
  2. git, hexo 관련 패키지 설치 ctrl + ,(comma) 단축키로 setting에 들어간 후 install에서 패키지를 검색해서 install한다.(git-plus, atom-hexo, hexo-front-matter)
