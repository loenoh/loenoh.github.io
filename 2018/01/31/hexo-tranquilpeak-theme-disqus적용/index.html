Hexo Tranquilpeak Theme Disqus적용

Jan 31, 2018 in hexo

Hexo 블로그에서 Disqus를 사용해보자.

Disqus는 블로그 포스트 하단에 댓글을 달 수 있게 해주는 기능이다.

먼저 Disqus 사이트에서 가입을 해야한다.

가입을 한 후 Home > setting > Admin > General 메뉴에 가면 다음 화면이 나타난다.

Add버튼을 클릭하면 댓글을 달 사이트를 등록하는 화면이 나온다.

사이트를 만들고 나면 disqus를 설치할 사이트가 어떤 플랫폼인지 선택하라는 페이지가 나온다.
해당 플랫폼이 없으면 하단부에

I don't see my platform listed, install manually with Universal Code. 라는

버튼을 클릭하면 소스에 직접 삽입할 수 있는 코드가 나타난다.

Hexo블로그는 플랫폼에는 없지만 왼쪽 Configure Disqus로 바로 넘어가도 무방하다.

Website Name, Website URL, Category, Description, Language를 세팅하고 마치면 된다.

세팅이 끝난 후 Admin 페이지에 가보면 Shortname 이 보일 것이다.
이것은 unique하고 변경도 되지 않는다. 만약 다른것을 사용하고 싶으면 다시 추가해야 된다.

그러면 Hexo 블로그 테마 환경 설정 파일 _config.yml에 추가하면된다.

disqus_shortname: [추가된 해당 사이트의 shortname]

이렇게 세팅하고 포스팅 하단을 확인하면 댓글을 달 수 있는 부분이 추가되어 있을 것이다.
